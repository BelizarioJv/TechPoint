🚀 TechPoint

TechPoint é uma aplicação web moderna de e-commerce desenvolvida como uma Single Page Application (SPA), com foco em performance, experiência do usuário e interface limpa.

🧠 Sobre o Projeto

Este projeto simula uma loja de tecnologia onde o usuário pode navegar entre produtos de forma rápida e fluida, sem recarregamento de página.

Foi desenvolvido com o objetivo de praticar e demonstrar habilidades em desenvolvimento front-end moderno, com foco em:

Componentização
Organização de código
Responsividade
Navegação no lado do cliente (client-side routing)

⚙️ Tecnologias Utilizadas:
React.js
React Router DOM
Hooks do React (useState, useEffect, useContext etc.)
Tailwind CSS

✨ Funcionalidades

⚡ Navegação rápida com arquitetura SPA
🔄 Rotas dinâmicas com React Router
🧩 Componentes reutilizáveis
📱 Layout responsivo
🎨 Interface moderna com Tailwind CSS
